serve.m platform is a quad in-order cores SoC implemented on VCU128, based on RISC-V ISA. It will be used in the FPGA cloud service (SERVE).
It is designed for hardware simulation of accelerators.
It's key features are HBM and network based booting.
The quad RISC-v cores (Rocket) can boot OpenSBI + u-boot.
The performance of HBM is measured, and the instructions about how to use HBM is provided.
